Data Cleaning with pandas
Selected 10 useful columns, removed duplicates using drop_duplicates() on pl_name — reduced from 39,443 to 6,128 unique planets. Identified missing values with isnull().sum(). Fixed sy_dist from TEXT to FLOAT using pd.to_numeric().

Analysis with SQLite
Imported data into SQLite via df.to_sql(). Wrote 6 queries using COUNT, GROUP BY, WHERE, AVG, ROUND, HAVING and Subqueries. Created a second table exoplanets_systems to analyze multi-planet star systems. Found KOI-351 holds the record with 8 confirmed planets.
