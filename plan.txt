Installation:

Cellule 8:
create_engine -> crée une connexion vers une base de données SQLite.
Le fichier exoplanets.db est crée automatiquement dans le dossier data/
df.to_sql -> prend le DataFrame pandas (6128 planètes nettoyées) et
l'insère dans une table SQL appelée exoplanets. C'est comme si on
importait le CSV dans une vraie base de données.

Pourquoi faire ?
Parce qu'en entreprise les données sont rarement dans des CSV, elles
sont dans des bases de données. En faisant ça, on montre que l'on sait
travailler avec SQL.

Cellule 9:
Combien de planètes par méthode de détection ? (detection_methods.png)
- Transit domine largement avec 4511 planètes - c'est la méthode du télescope Kepler
- Radial Velocity en 2ème avec 1171 - la méthode historique
- Microlensing Imaging... sont des méthodes rares et difficiles
