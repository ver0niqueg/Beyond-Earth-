Download table au CSV format

Colonnes importantes

pl_name = name of the planet
disc_year = the year it has been discovery
discoverymethod = the discovery method
pl_rade = rayon (terrestrial)
pl_masse = mass (terrestrial)
pl_orbper = orbital period (days)
sy_dist = parsecs distance
pl_eqt = temperature d'equilibre (K)
st_teff = temperature of the host star
sy_snum = number of stars in the system
